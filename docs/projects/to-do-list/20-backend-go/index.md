# Backend - Go

Welcome to the To Do List Backend documentation. This service is built using [Go](https://go.dev/).

## Technology Stack

- **Language**: Go
- **Framework**: Standard Library / Gin or Echo (TBD)
- **Database**: PostgreSQL with SQLC or GORM
- **Authentication**: JWT

## Getting Started

### Prerequisites

- Go 1.21+
- Docker (for database)

### Running the App

```bash
go run cmd/server/main.go
```
