# Frontend Web - Next.js

Welcome to the To Do List Frontend documentation. This application is built using [Next.js](https://nextjs.org/).

## Technology Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS / CSS Modules
- **Icons**: Lucide React

## Getting Started

### Prerequisites

- Node.js 18.x or later
- pnpm

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

## Project Structure

- `app/`: Contains routes and pages.
- `components/`: Reusable UI components.
- `styles/`: Global and component-specific styles.
- `lib/`: Utility functions and API helpers.
